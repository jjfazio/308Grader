Acceptance Test Plan: Adding a Category

The test plan for adding a Category to a spreadsheet course is as following:

Case No. User Inputs and Commands Expected Output Remarks                   Results                  
1 Navigate to Menu->Assignments->Add Category

Leave all text entry fields blank and select the "Save" button. 		An "Invalid Student Input" error dialog should appear, informing the user to choose a parent Category. Tests adding new Category with completely blank text-entry fields, and not choosing parent category Pass
2 Navigate to Menu->Assignments->Add Category

Leave all text entry fields blank but select the Overall Category. Then select the "Save" button. 		An "Invalid Student Input" error dialog should appear, informing the user to choose a name and percent of parent for the new Category. Tests adding new Category with completely blank text-entry fields, but choosing parent Category Pass
3 Navigate to Menu->Assignments->Add Category

Fill the name field with "Midterms" and percent of parent field with "two" Then select the "Save" button. 		An "Invalid Student Input" error dialog should appear, informing the user that percent of parent can only be a number. Tests adding new Category filled in name field, but invalid Percent of Parent field. Pass
4 Navigate to Menu->Assignments->Add Category

Fill the name field with "Midterms" and percent of parent field with 30 Then select the "Save" button. 		The Add Category dialog closes. Confirm the Category that was just created is now displayed in the spreadsheet underneath its parent category along with its percent of parent. Tests adding new Category filled in name field, and valid percent of parent field Pass
5 Navigate to Menu->Assignments->Delete Category

Select the "Delete" button without choosing any category. 		An "Invalid Input" error dialog appears, informing the user that the user needs to select a category to delete. Tests deleting a category without selecting any category Pass
6 Navigate to Menu->Assignments->Delete Category

Select the Overall from the list. Then select the "Delete" button. 		An "Invalid Input" error dialog should appear, informing the user that the Overall category cannot be deleted. Tests deleting the Overall category. Pass
7 Navigate to Menu->Assignments->Delete Category

Select any category except the Overall category. Then select the "Delete" button. 		The Delete Category dialog closes. Confirm the selected category is removed from the the categories column. Tests deleting the Overall category. Pass
8 Navigate to Menu->Assignments->Edit Category

Do not select any category, and select the "Edit" button. 		An "Invalid Input" error dialog appears, informing the user that the user needs to select a category other than Overall to edit. Tests Editing a category with invalid input. Pass
9 Navigate to Menu->Assignments->Edit Category

select any category except Overall, and select the "Edit" button. "Name" and "Percent of Parent" fields will become enabled and filled with the selected category's information. Then click "Save" button. 		An "Invalid Input" error dialog appears, informing the user that the user needs to select a parent category. Tests Editing a category without choosing the new parent category. Pass
9 Navigate to Menu->Assignments->Edit Category

select any category except Overall, and select the "Edit" button. "Name" and "Percent of Parent" fields will become enabled and filled with the selected category's information. Fill the "Name" and "Percent of Parent" fields with a String and number. Select a new parent category. Then click "Save" button. 		The Edit Category dialog closes. Confirm that the Name and Percent of Parent of the category has changed in the spreadsheet. Tests Editing a category with valid inputs. Pass
