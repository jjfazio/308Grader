Acceptance Test Plan: Deleting a student

Author: Kevin Feutz

The test plan for deleting a student in a spreadsheet course is as following:

Case No. User Inputs and Commands Expected Output Remarks                   Results                  
1 Make sure you have students enrolled in at least one course

Navigate to Menu->Students->Delete Student

Select the first person that appears on the list and click on the "Delete Selected" button

Select the course in the next dialog that appears and click "Delete".

A confirmation dialog should appear, select "Yes" 		The dialog should close. Check the roster that student was removed from the course you selected. Tests deleting a student from one course. Pass
2 Make sure you have a student enrolled in two courses

Navigate to Menu->Students->Delete Student

Select the student who is enrolled in two courses and click on the "Delete Selected" button

Select both courses in the next dialog that appears and click "Delete".

A confirmation dialog should appear, select "Yes" 		The dialog should close. Check both rosters that student was removed from the courses you selected. Tests deleting a student from multiple courses. Pass
3 Make sure you have students enrolled in at least one course

Navigate to Menu->Students->Delete Student

Select the first person that appears on the list and click on the "Delete Selected" button

Don't select any courses and select "Delete". 		A warning dialog should appear stating that courses must be selected if the "Delete" button is clicked. Tests deleting a student from no courses. Pass
