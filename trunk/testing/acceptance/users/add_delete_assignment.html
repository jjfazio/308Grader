Acceptance Test Plan: Adding a student

The test plan for adding a student to a spreadsheet course is as following:

Case No. User Inputs and Commands Expected Output Remarks                   Results                  
1 Navigate to Menu->Assignments->Add Assignment

Leave all text entry fields blank and select the "Save" button. 		An "Invalid Input" error dialog should appear, informing the user to select a Category for the Assignment. Tests adding new student with completely blank text-entry fields and without choosing parent category. Pass
2 Navigate to Menu->Students->Add Assignment

Select the parent category
In the name field, enter ""
In the weight field enter "ten"
In the due date field enter "13/1/2013""
In the points field enter 200 		An "Invalid Input" error dialog appears, stating that the name, weight, due date, and points values are invalid. Tests adding new Assignment with invalid inputs. Pass
3 Navigate to Menu->Students->Add Assignment

Select the parent category
In the name field, enter "Midterms"
In the weight field enter 30
In the due date field enter "12/13/2013""
In the points field enter 50 		The Add Assignment dialog closes. Confirm the new assignment is added to the Assignments column of the spreadsheet. Tests adding new Assignment with valid input. Pass
4 Navigate to Menu->Students->Delete Assignment

Select a category and select "Delete" button. 		An "Invalid Input" error dialog appears, stating that an assignment should be selected to be deleted. Tests removing a category in remove assignment page. Pass
5 Navigate to Menu->Students->Delete Assignment

Select an assignment and select "Delete" button. 		The Add Assignment dialog closes. Confirm the assignment is removed in Assignments column of the spreadsheet. Tests removing an assignment. Pass
