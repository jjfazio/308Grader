Acceptance Test Plan: Editing a student

The test plan for editing a student in a spreadsheet course is as following:

Case No. User Inputs and Commands Expected Output Remarks                   Results                  
1 Make sure you have students enrolled in at least one course

Navigate to Menu->Students->Edit Student

Select the first person that appears on the list and click on the "Edit Selected" button

An edit dialog should appear, change the student's first name to "Jones", and select "Confirm Edit" 		The dialog should close. Check the roster that the previous student was located and make sure his or her new first name is "Jones" Tests editting a student's first name. Pass
2 Make sure you have students enrolled in at least one course

Navigate to Menu->Students->Edit Student

Select the first person that appears on the list and click on the "Edit Selected" button

An edit dialog should appear, change the student's first name to "#$%Jones", and select "Confirm Edit" 		An error dialog should appear, stating "First name must only contain alphabetic characters," and no change to the student selected should occur. Tests editting a student's first name with invalid input. Pass
3 Make sure you have students enrolled in at least one course

Navigate to Menu->Students->Edit Student

Select the first person that appears on the list and click on the "Edit Selected" button

An edit dialog should appear. Select the course in the "Course(s) Enrolled" dialog, and select "Delete Course"

Click on the "Add Course" button, and add a different course than the one the student was previously enrolled in

Select "Confirm Edit". 		The student that was editted should now no longer appear in the original course, but should appear in the newly added course. Tests editting a student's list of courses enrolled. Pass
4 Make sure you have students enrolled in at least one course

Navigate to Menu->Students->Edit Student

Select the first person that appears on the list and click on the "Edit Selected" button

An edit dialog should appear. Clear all text fields on this dialog

Select the "Confirm Edit" button. 		An error dialog should appear, indicating which fields are not allowed to be blank. Tests editting a student with all empty fields. Pass
