Acceptance Test Plan: Viewing Graphs and Adjusting Curves

Author: Erik Owen

The test plan for viewing graphs and adjusting the curves for an assignment/category is as following:

Case No. User Inputs and Commands Expected Output Remarks                   Results                  
1 Make sure you are in a Gradebook which has assignments, categories, and students, with student grades for the respective assignments and categories. Look ad the grading.html file in acceptance/spreadsheet for information on this.

Navigate to Menu->View->View Graphs

Click on Overall category, and press select.

 The 'Graph and Adjust Curve' Page should appear. The graphs data should correctly reflect the student's scores for that particular category. Tests graphs dynamically being created to reflect students scores for category showing. Pass
2 While still on the 'Graphs and Adjust Curve' page, go into the add percent curve text box in the 'Add Curve' box

in the text box, enter "10"
Press the 'Save Curved Grades' Button
Exit out of the Graph and Adjust Curves page to go back to the main spreadsheet view.
 All of the student's scores for the overall grade should have increased by 10%. Tests adding a standard curve functionality. Pass
3 Navigate to Menu->View->View Graphs

Click an Assignment of your choice, and press select.

 The Graph and 'Adjust Curve' page should appear. This time, the graph's data should correctly reflect the student's scores for that particular assignment. Tests viewing the graphs of the assignment to dynamically show students scores. Pass
4 While still on the 'Graphs and Adjust Curves' page, go the 'View Grades As' box and uncheck pie chart

Click the radio button that says 1% granularity. 		The pie chart should be hidden from the view, and the bar chart's data should have changed from 10% intervals to 1% intervals. (Note, sometimes the 1% data does not appear in the bar chart if there is a lot of data, because JavaFx needs a certain amount of room to display all the data, and will not show any data if there is not the needed amount of room.) Tests hiding graph types and changing interval granularity. Pass
4 While still on the 'Graphs and Adjust Curves' page, go the 'Adjust Curve' box and click the 'Add Custom Curve' button.

go into the table and change the low value for the 'A' grade column to 82. Press the 'Refresh Graphs' button. 		The 'Custom Curve Adjuster' page should have appeared which allows users to edit the low and high values of the different grade ranges. After the user hit the Refresh Graphs button, an error dialogue should appear that tells the students that there are overlapping grade ranges. Tests the 'Custom Curve Adjuster' page, and the functionality to determine if the user has two grade ranges that intersect. Pass
4 While still on the 'Custom Curve Adjuster' page, go to the 'B' grade column and change the high to be 81, so that B's range does not overlap with A. Hit the 'Refresh Graph' button, then hit The 'Apply Curve' button. After the user hit the 'Refresh Graph' button, the graph should have changed to reflect the new distribution of students with the grade range in the table. After the user saved these GradeRanges by pressing the 'Apply Curve' button, the 'Custom Curve Adjuster' and 'View Graphs and Adjust Curves' pages should be disappear. Back in the main spreadsheet, when the user changes a student's grades in the table, the new grade ranges will be reflected in the 'Overall' column. Tests input new ranges in the 'Custom Curve Adjuster' page, tests saving the new desired grade ranges. Pass
