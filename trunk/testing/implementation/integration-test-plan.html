Grader Tool Integration Test Plan


After the classes in each package are independently tested with mock student and course data, perform integration testing in the following phases:

  1. Integrate the gradebook package with the spreadsheet package and rerun the class tests in the gradebook package.

  2. Add the view package to the integrated spreadsheet + gradebook packages; rerun the class tests in the gradebook and view packages.

  3. Add the users package to the integrated spreadsheet + gradebook + view; rerun the class tests in the gradebook, users, and spreadsheet packages.

  4. Integrate the assignments_categories and package and rerun the class tests in the spreadsheet, users, assignments_categories package.

  5. Add the file package to the integrated spreadsheet + users + assignments_categories; rerun the class tests in the assignment_categories, users, spreadsheet, and file packages.

  6. Add the mock student and course data to the integrated spreadsheet + assignment_categories + users + file + view; rerun the class tests in the spreadsheet, view, assignment_categories, and users packages.

  7. Add the graph package to the integrated spreadsheet + view + users + assignment_categories; rerun the class tests in the spreadsheet, view, users, assignment_categories, and graph packages.

  8. Add the external package to the integrated spreadsheet + view + users + assignment_categories + file + graph; rerun class tests in the spreadsheet, view, users, assignment_categories, file, graph and external packages.
