2.3. Manage Students

Scenarios in this section illustrate functions available to the teacher or teacher's assistant, depending on permissions. The following functions are allowed:

  • Add a student
  • Edit a student's information
  • Delete a student
  • View students' information

The functions listed above manipulate data in the user's class rosters. The sections in 2.3 manipulate data existing in the class roster as shown in Figure 2.2.2.D. Please refer to Figure 2.2.2.D Resulting spreadsheet view for the initial class roster. This represents the class roster before any Manage Students operations are used.
