1.5.3. Flubaroo

Flubaroo is a grading tool integrated with Google Docs. While not the most popular grading tool on the market, Flubaroo is simple to use and has many handy features. Flubaroo is free to the public, and can be installed and ready to use in a matter of minutes.

Overall, Flubaroo is a subpar grading tool. Students do not have an interface to view their graded work, and Flubaroo relies on Google Docs to store the grades.

Good Features:

  • Simple to install and run
  • Handy if Google Docs is already used to store grades
  • Sends email to students notifying them of the grade they received on an assignment
  • A report is generated and available to the grader after each assignment is done being graded
  • Flubaroo is available in Spanish

Bad Features:

  • Students only way of seeing grades is through email notifications
  • Only visual representation of grade distribution is a simple histogram in the grader's report
  • Flubaroo sends an email using your email address (some may find this concerning)

Missing Features:

  • No way of importing grades besides using Google Docs
  • No way of storing cumulative grades for the quarter
  • No way of offering extra credit
