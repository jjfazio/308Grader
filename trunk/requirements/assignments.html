2.2.2 Adding / Editing Assignments

When the user select the "Assignment" item from the "Item" menu he/she can add, edit, and delete assignments and categories of assignments in the following dialogue.

Adding, Editing, and Deleting Assignments.

The user can add an assignment by clicking on Adding new assignment under "Name" drop down list. If the user chooses a priviously created assignemt, the software will fill out all of the fields of this page with the data that user chose for that assignment when he/she created it. Then, the user will have the choise to change the attributes of that assignment.

Similarly, the instructor can create, delete, or choose a previously created catagory for the assignment at hand by using the drop down list labeled "Category". Additionally the instructor should choose they grading scale for that assignment from the "Grade Type" drop down list. Finally, the user should specify the due date on the calendar and the weight of the assignment in the text box labeled "Points".


The scenarios in this section give the user the ability to create new assignments by providing the following functionalities.
  • Adding new assignment.
  • Adding new category of assignments.
  • Deleting category of assignments.
