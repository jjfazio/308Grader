1.5. Related Systems

Similar commercial systems to the Grader Tool are:

  • Aeries
  • Easy Grade Pro
  • Flubaroo
  • Microsoft Excel
  • PowerTeacher
  • ThinkWave

The systems listed above have a wide spectrum of grading tools and features. In terms of such functionality, the goal for the Grader Tool is to provide the important core features found in these tools, not the full range of features found in all of them. The Grader Tool does provide modest functional improvements in four areas, when compared to most commercially available tools. These improvements are:

  1. Simple grade spreadsheet with ability to add, modify, or preset content

  2. Give the student a presentable interface to view grades, class grades, and ability to see what future grades they need to get on assignments to get a certain grade in the class

  3. Graphical summaries of the class grade breakdown

  4. Grader has ability to curve grades or assignments

The goals for the Grader Tool relate to creating an easy to use interface with features not met by any of the above commercial systems.
