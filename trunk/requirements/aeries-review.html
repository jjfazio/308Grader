1.5.1. Aeries Student Information System

Aeries Student Information System is a software system developed by Eagle Software. It is targeted at California education institutions, especially high schools, but its grading functionally is similar to a university level system. Aeries does not provide a stimulating or intuitive visual interface, but functionally it is solid. It is not a stand-alone application, but rather a browser accessible page.

Good Features:

  • Real time data access and entering
  • Secure administrator, student, and teacher access
  • Simple UI allows for lots of data on screen
  • Calendar view of assignments
  • Student search bar
  • Transcripts show students’ history

Bad Features:

  • Switching between “spreadsheets” (no tabs)
  • Clunky “My Tasks”
  • Column widths default to very narrow
  • Labels hard to read

Missing Features:

  • Extra Credit functionality
  • Visualization (graphs, charts, etc.) of grades
  • TA user functionality
