2. Functional Requirements

The primary functionality of the Grader to is to recored and view student's grades. The Grader Tool is used both by the grader (teacher or TA), as well as the students. However, the two different user types have different views and different permissions. Not only is the tool used for setting and viewing grades, but it can also perform tasks like importing assignments, adjusting curves, and viewing overall statistics about a particular assignment, a student, or a class as a whole.

Following an overview of the Grader Tool's user interface, details of system use are presented in the following scenarios:

  • the grader's (teacher's or TA's) perspective of the Grader Tool,
  • and the student's perspecitve of the Grader Tool.
These scenarios cover all functional requirements for the system.
