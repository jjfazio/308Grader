1.5.6. ThinkWave

ThinkWave is a free online grading tool. The ease of use is attractive for simple grading, making it an excellent tool for elementary and middle school teachers. Thinkwave has a nice graphical interface with straightforward navigation. The Grading tool requires more of a sophisticated and customizable grading procedure. Most importantly ThinkWave lacks grade curving functionality and statistical breakdowns of assignments.

Overall, Thinkwave provides a decent level of grading functionality with a clean interface.

Good Features:

  • Simplistic interface, easy to use
  • Has basic grading functionality
  • Ability to set weights for categories
  • Can add students and edit their information
  • Able to choose the types of points for assignments (percentage, pass/fail, letter grade)
  • Students can turn stuff in online
  • Attendance support

Bad Features:

  • Able to edit grades from any computer (potential security issue)
  • The way extra credit gets assigned is confusing
  • Spreadsheet needs to be larger and contain more information
  • Students can only view their grades
  • Cannot have multiple people edit grades for one class

Missing Features:

  • Statistical breakdowns of assignments/categories/final grade
  • Graphical visuals
  • Ability for a student to calculate what they need to get on assignments for a certain grade
  • Late Policy
  • Importing students from a file
  • Setting a curve for assignments/categories/final grade
  • Connection to the Cal Poly registrar
