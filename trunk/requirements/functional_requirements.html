2. Functional Requirements

The following sections describe the functional requirements for the grader tool.

  • Section 2.1 - User Interface Overview - gives an overview of the three different users, and the initial home screens.
  • Section 2.2 - Basic Gradebook Setup - shows how to initially set up the gradebook.
  • Section 2.3 - Manage Students - shows how to add/edit/delete students from a class.
  • Section 2.4 - Manage Assignments/Categories - shows how to add/edit/delete assignments and categories for a class
  • Section 2.5 - Using Spreadsheets - shows how to enter grades and general spreadsheet usage
  • Section 2.6 - View Detailed Statistics/Graphs - shows how to view graphs and adjust class class/assignment curves.
  • Section 2.7 - Student Functions - shows the tool from the perspective of a student.
